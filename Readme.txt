This project will get the top 250 movies list from the IMDB site with the details as Movie Name, Year and the ratings and load it to the database. User can also fetch the loaded data.

Pre- requsite to run the project
1) Locate the IMDBMovies folder in C drive .
2) All the paths are configured as C://IMDBMovies//.... So, don't make any changes in the paths.
3) Import all required jars are located in the 'Required jar' folder and subfolders.
4) Run the project from testng.xml as a textng suite
